# To-Do

To-Do is a Task Management app developed in Flutter

## Features
* Create/Edit/Delete Tasks
* Add Sub-tasks under each Task
* Modify the state of each Sub-task as done/undone
* Save Tasks on the Cloud using Cloud Firestore
